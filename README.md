# Finance Core Toolkit 🛠️

Ce répertoire contient le moteur de calcul financier et les outils de base pour le pricing d'options, l'optimisation de portefeuille et l'analyse de marché.

## 📦 Contenu du dépôt

- **Pricers d'Options** : Black-Scholes (Standard & Merton), Cox-Ross-Rubinstein (Binomial), Trinomial.
- **Optimisation de Portefeuille** : Optimisation par Monte Carlo et méthodes analytiques.
- **Market Tools** : Scripts de récupération de données boursières et snapshots de marché.

## 🚀 Utilisation

Chaque script peut être exécuté individuellement. Par exemple :

```bash
python3 market_snapshot.py
```
